# What is Hemslay

Just a simple CRUD API to manipulate data in Go

### Modules

- GIN
- GORM
- GODOTENV
- Azure SQL Server
- CompileDaemon

### Initialize with CompileDaemon

For live-changes purpouses

```go
CompileDaemon -command="./hemslay-api"
```
